# Build a Client Site on This Template

The canonical, tool-neutral instructions live in `AGENTS.md` at the repo root — a single source of truth shared with other coding agents (Codex, Cursor, OpenCode, Antigravity).

**Read `AGENTS.md` now and follow it exactly.** It contains:

- The hard rules protecting the contact form's API contract (never violate these)
- What you may freely customize (styling, pages, nav, copy)
- The build workflow, ending with an `npm run build` verification
- The one sanctioned procedure for adding custom form fields
- The definition of done

Do not proceed with any site-building work until you have read `AGENTS.md` in full.
